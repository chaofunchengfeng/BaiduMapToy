# BaiduMapToy 百度地图工具

A Chrome extension(mv3) for [Baidu Map https://map.baidu.com/](https://map.baidu.com/)

百度地图Chrome扩展（mv3）

查看全景后，自动在地图标注全景位置，以便知晓哪里被查看过。即：全景查看的历史记录在地图上显示。

## 使用

查看全景后，自动标注。

点击图标清空历史记录。
